# Boxing Interval Timer

A professional combat sports timer for iPhone and iPad, built with SwiftUI.

## Features

- **Fight Timer** - Presets for Boxing, MMA, K1, Muay Thai, BJJ, Judo, Wrestling, Taekwondo
- **Interval Training** - HIIT workouts for running, treadmill, air bike, and bag work
- **Custom Profiles** - Create and save your own timer configurations
- **Workout History** - Track all completed workouts with date and duration
- **Statistics** - View streaks, total time, favorite sport, and weekly progress
- **Todo List** - Plan your training sessions
- **Live Activity** - Timer visible on the lock screen
- **6 Languages** - German, English, Arabic, Spanish, French, Russian

## Tech Stack

- Swift / SwiftUI
- StoreKit 2 (In-App Purchases)
- Core Data (workout history)
- WidgetKit (Live Activity)
- MVVM architecture
